## What's Included

<CardGrid stagger>
  <Card title="Clean Architecture" icon="puzzle">
    Layered package design: RPC (API) -> Core (Use Cases + Authorization) -> Repository (Data Access) -> Database. Each layer has a single responsibility.
  </Card>
  <Card title="oRPC + TanStack Query" icon="rocket">
    End-to-end type-safe RPC with automatic cache management. Server-side prefetching with client hydration. No manual API clients or query keys.
  </Card>
  <Card title="Clerk Authentication" icon="lock">
    Managed authentication abstracted into a workspace package. Server/client utilities, middleware proxy, and themed sign-in/sign-up components.
  </Card>
  <Card title="Drizzle ORM + Neon" icon="setting">
    Type-safe database with serverless PostgreSQL. Automatic Zod schema generation via drizzle-zod. Migrations, seeding, and Studio GUI.
  </Card>
  <Card title="Stripe Payments" icon="star">
    Subscriptions, checkout sessions, billing portal, and webhook handling. Pricing configuration with plan management utilities.
  </Card>
  <Card title="Turborepo Monorepo" icon="seti:config">
    Optimized builds with task pipelines, workspace dependencies, and shared TypeScript configurations. Bun package manager with catalog dependencies.
  </Card>
  <Card title="PostHog Analytics" icon="laptop">
    Client and server-side event tracking. Vercel Analytics for performance monitoring. Optional Google Analytics integration.
  </Card>
  <Card title="Sentry Observability" icon="magnifier">
    Error tracking, session replay, and OpenTelemetry tracing for oRPC procedures and Drizzle queries. No-op when envs are missing.
  </Card>
  <Card title="Typed Analytics Metrics" icon="rocket">
    First-class North Star, Activation, and Retention definitions in code. Typed `EVENTS` registry constrains every `capture()` call.
  </Card>
  <Card title="shadcn/ui + Tailwind v4" icon="star">
    27 Radix UI components with OKLch color system, dark mode, and sidebar theming. Tailwind CSS v4 with PostCSS.
  </Card>
  <Card title="Resend Email" icon="document">
    React Email templates with Tailwind styling. Transactional email delivery with type-safe send functions.
  </Card>
  <Card title="Trigger.dev Jobs" icon="sun">
    Background task execution with retry configuration. Scheduled tasks and one-time jobs with the Trigger.dev SDK.
  </Card>
  <Card title="Vitest + Playwright" icon="magnifier">
    Unit testing with Vitest and comprehensive E2E testing with Playwright. Coverage reports and UI mode.
  </Card>
</CardGrid>
